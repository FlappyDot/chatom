# Chatom

轻量 PHP 聊天室，无需数据库，解压即用。

## 功能

- 多人在线聊天，多房间
- `default` 为公开房间，新建房间可设密码（留空自动生成）
- 支持图片和常见文件附件上传（单个文件最大 10MB）
- 消息经 Base64 + 字符替换编码传输，可防日志明文记录，**不提供真正的加密保护**
- 消息 7 天后自动清除
- 昵称本地保存，刷新不丢失
- 响应式，桌面双栏 / 移动端单栏

## 部署

1. 解压到服务器目录
2. 创建 `chat_data/` 目录并给予 PHP 写入权限
3. 修改 `config.php` 设置站点标题和 `logoUrl`（可选）
4. 访问 `index.php`

```
├── index.php
├── config.php
└── chat_data/
```

首次上传附件时会自动创建 `chat_data/uploads/`，同样需要 PHP 具备写入权限。

## 关于 Chatom

名称来源：

- **Chat** — 聊天
- **Atom** — 象征轻量、极简，如原子般小巧
